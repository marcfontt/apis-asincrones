# Search Component

Pàgina de busca global del portal (integrada amb Backstage Search).

## Arxius

- **SearchPage.tsx**: Renderitza la interfície de busca. Permet buscar
  components del catàleg, escenaris, resultats anteriors.

## Ús

Accessible a `/search` o via la barra de busca de la top nav.

## Capacitats

- Busca en temps real entre components i escenaris.
- Filtra per categoria (architecture, protocol, platform).
- Mostra resultats destacats del Backstage search backend.
