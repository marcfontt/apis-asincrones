# `scripts/` — Scripts auxiliars

Petits scripts que no formen part del codi de l'aplicació però ajuden
en operacions del dia a dia: build, smoke tests, neteja, etc.

Vegeu el contingut de la carpeta per a la llista completa. La majoria
es poden executar des de l'arrel del repositori; cada script té el
seu propi shebang i mostra ajuda amb `--help` quan és aplicable.

També tenim el pipeline principal a [`../deploy-all.sh`](../deploy-all.sh).
